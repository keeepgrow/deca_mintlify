# DECA Project Setup Guide

## Getting Started

To start using DECA, you need to create a project. Once you complete all fields in the project form, the agent will automatically start a conversation with you for content creation.

## Project Form Fields

### 1. Project Name (Required)

**Purpose**: Project identification and management

**How to write**:

```wrap
[Brand Name] - [Content Type] - [Period]
Example: "Notion - Productivity Guide - 2026 Q1"
```

---

### 2. Brand (Required)

**Purpose**: Foundation for DECA to analyze and learn about your brand

DECA analyzes your selected brand's website, products, and tone to create Brand Context Memory. This information is stored and automatically applied to all content creation.

**How to write**:

- Select an existing brand or click "<u>\+ Add New Brand</u>"
- For new brand: Enter brand name, website URL, and brief description

---

### 3. Project Goal (Required)

**Purpose**: Content strategy direction and Target Prompts selection criteria

DECA finds relevant customer questions and determines content structure based on this goal.

**Options**:

- **Increase Brand Visibility**

  Create content to increase brand exposure and awareness. DECA prioritizes discovery-focused Target Prompts like "[category] recommendations" and "best [product]."
- **Build Topical Authority**

  Establish expertise and credibility in specific topics. DECA identifies Target Prompts for in-depth guides, industry insights, and trend analysis that demonstrate expertise.
- **Drive Conversions**

  Focus on lead generation and purchase conversions. DECA prioritizes buying-decision Target Prompts like "comparison," "vs," "pricing," and "free vs paid."
- **Others (Specify custom goal)**

  If you have a specific goal not listed above, enter it here. DECA will develop a content strategy tailored to your custom goal.

---

### 4. Country & Language (Required)

**Purpose**: Regional search patterns and language optimization

The same question is expressed differently across countries. DECA reflects actual search patterns in your selected country.

**How to write**:

- Select country and language of your target market
- For multi-country strategy, create separate projects per country (recommended)

---

### 5. Target Audience

**Purpose**: Core input for DECA's Persona analysis and Target Prompts discovery

The more specific, the more accurate Target Prompts DECA can identify.

**Writing formula**:

<Callout icon="circle-info" color="#B7A3FF">
  [Who] + [What they do] + [Why interested] + [How they decide]
</Callout>

**Example**:

```
"Startup founders aged 25-40 managing teams of 10-50 people, 
working in remote/hybrid environments. They want to solve 
inefficiencies from switching between multiple tools. They 
prioritize ROI, test with free trials before adoption, and 
trust colleague recommendations and online reviews."
```

---

### 6. Content Guidelines

**Purpose**: Rules DECA must follow when generating content

- For legally sensitive topics (e.g., finance, healthcare, personal data), clearly specify relevant legal information and regulatory compliance requirements in the guidelines.

**Required items**:

**1. Must Include**

<Callout icon="circle-info" color="#B7A3FF">
  Example: Include data with all claims, minimum 2 real use cases
</Callout>

**2. Never Include**

<Callout icon="circle-info" color="#B7A3FF">
  Example: No direct competitor bashing, no unverifiable superlatives
</Callout>

**3. Tone & Voice**

<Callout icon="circle-info" color="#B7A3FF">
  Example: Professional but friendly, explain technical terms simply
</Callout>

**Simple example**:

```
Must Include:
- Real customer cases
- Statistics with sources

Never Include:
- Competitor bashing
- Exaggerations like "best", "perfect"

Tone:
- Professional but accessible
- Write as if speaking directly to reader
```

---

## After Project Creation

Once you complete the project form, **the agent automatically starts a conversation** and guides you through the next steps.

Based on your project information, the DECA agent will:

- Conduct Brand Research
- Discover Target Prompts
- Propose content strategy
- Draft content

All processes are guided through conversation.
