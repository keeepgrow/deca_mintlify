# Add Your Brand

Before creating a project, you need to add your brand first. DECA analyzes your brand's website to automatically identify positioning, differentiators, and authority signals. This information is used for all content generation.

## How to Add a Brand

There are two ways to add a brand.

### Method 1: From the Brands Page

Click **Brands** in the left menu, then click the **"+ Add Brand"** button in the top right. This is a good way to add and manage multiple brands in advance.

### Method 2: During Project Creation

When creating a project, select **"+ Add Brand"** at the brand selection step. You can set up both the project and brand at once.

## Enter Brand Information

You only need to enter two pieces of information:

**Brand Name** Enter the official brand name as you want AI search engines to recognize and cite it.

```
Good examples: Notion, Slack, Stripe
Bad examples: Notion docs, Slack Inc., stripe.com
```

**Brand Website URL** Enter your brand's official website address. DECA will analyze this site to learn your brand context.

```
https://notion.so
https://slack.com
https://stripe.com
```

## Generate Brand Research

Click the **"Generate Brand Research"** button and DECA will automatically analyze your website. This takes about 1 minute and includes:

- Getting Brand Research Template
- Searching Citation Sources
- Writing File
- Completion

Once the analysis is complete, you can review the results in the Brand Info and Brand Research tabs.

## Review and Refine Results

After Brand Research is complete, review what DECA has identified:

- Product/service description
- Brand positioning
- Key differentiators
- Tone and voice

If the content isn't accurate or needs refinement, you can edit it directly. This information is used for all content generation, so it's important to set it up accurately from the start.

## Manage Brands

You can view all added brands on the Brands page:

- **Name**: Brand name
- **Website**: Registered website
- **Projects**: Number of projects created with this brand
- **Last Updated**: Last modification time
- **Research**: View and edit Brand Research

Click the **Research** link for any brand to view and edit its Brand Research at any time.

## Next Steps

Once your brand is added, you can create projects. When creating a project, select your brand and DECA will remember the brand context to generate consistent content.

## FAQ

<AccordionGroup>
  <Accordion title="Can I add multiple brands?" icon="circle-question">
    Yes, you can add each brand you manage separately. Each brand has its own independent Brand Research.
  </Accordion>
  <Accordion title="Can I edit brand information later?" icon="circle-question">
    Yes, you can view and edit Research from the Brands page at any time. Changes will be reflected in content generated afterwards.
  </Accordion>
  <Accordion title="What if I don't have a website? " icon="circle-question">
    Enter a landing page, LinkedIn page, or any public page URL that describes your product. DECA analyzes publicly accessible information.
  </Accordion>
</AccordionGroup>
